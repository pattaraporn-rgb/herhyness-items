# ค้นหาสินค้า HER HYNESS 🔍

เว็บค้นหาสินค้า: พิมพ์รหัสสินค้า (บาร์โค้ด / รหัสร้าน / ERP) หรือชื่อใบเสร็จ แล้วดูว่าคือ Search Name อะไร

เปิดใช้งานออนไลน์: `https://<ชื่อผู้ใช้>.github.io/herhyness-search/`

---

## ไฟล์ในโปรเจกต์

| ไฟล์ | คืออะไร | ต้องอัปขึ้น GitHub ไหม |
|---|---|---|
| `index.html` | หน้าเว็บ + ตรรกะการค้นหา | ✅ **ต้องอัป** |
| `products.json` | ข้อมูลสินค้า 265 รายการ (แก้ตรงนี้เวลาอัปเดต) | ✅ **ต้องอัป** |
| `Search Name.xlsx` | ฐานข้อมูลต้นฉบับ (source) | ไม่บังคับ เก็บไว้เป็นต้นทาง |
| `รหัส item herhyness.csv` | บันทึกการขาย ใช้เสริมรหัสที่ตารางแม่ขาด | ไม่บังคับ |

> ⚠️ **ต้องอัป `index.html` กับ `products.json` คู่กันเสมอ** เพราะหน้าเว็บจะไปโหลดข้อมูลจาก `products.json`
>
> ⚠️ **อย่าดับเบิลคลิกเปิด `index.html` จากเครื่องตรงๆ** — มันจะโหลดข้อมูลไม่ได้ (กฎความปลอดภัยของเบราว์เซอร์) ต้องเปิดผ่านลิงก์ GitHub Pages เท่านั้น

## วิธีนำขึ้น GitHub (ทำผ่านเว็บล้วนๆ ไม่ต้องลงโปรแกรม)

### ขั้นที่ 1 — สมัคร/เข้าสู่ระบบ
ไปที่ https://github.com แล้ว Sign in (ยังไม่มีให้กด Sign up สมัครฟรี)

### ขั้นที่ 2 — สร้าง Repository
1. มุมขวาบน กด **+** -> **New repository**
2. **Repository name:** พิมพ์ `herhyness-search`
3. เลือก **Public** (ต้อง Public ถึงจะเปิดเว็บฟรีได้)
4. กด **Create repository**

### ขั้นที่ 3 — อัปโหลดไฟล์ (อัปทั้ง 2 ไฟล์พร้อมกัน)
1. กดลิงก์ **uploading an existing file** (หรือ **Add file** -> **Upload files**)
2. ลากทั้ง **`index.html`** และ **`products.json`** เข้าไปพร้อมกัน
3. เลื่อนลงล่าง กด **Commit changes**

### ขั้นที่ 4 — เปิดเว็บ (GitHub Pages)
1. กดแท็บ **Settings** ด้านบน
2. เมนูซ้าย กด **Pages**
3. หัวข้อ **Branch** เลือก **main** และโฟลเดอร์ **/ (root)** กด **Save**
4. รอ 1-2 นาที แล้ว refresh หน้า
5. จะได้ลิงก์: `https://<ชื่อผู้ใช้>.github.io/herhyness-search/`
6. กดลิงก์ -> เปิดใช้ได้เลย ทุกเครื่อง ทุกมือถือ

> บุ๊กมาร์กลิงก์ไว้ที่หน้าจอมือถือ ใช้เหมือนแอปได้เลย

## วิธีอัปเดตข้อมูลภายหลัง (ข้อดีของการแยกไฟล์)

**แก้ข้อมูลเล็กน้อย** (เช่น เพิ่มสินค้า/แก้ชื่อ) ทำได้ที่ `products.json` โดยไม่ต้องยุ่งกับ `index.html` เลย:
1. บน GitHub เปิดไฟล์ `products.json` -> กดไอคอนดินสอ (Edit)
2. แก้ข้อความ (รูปแบบ JSON) -> **Commit changes**
3. เว็บอัปเดตเองใน 1-2 นาที

**อัปเดตจากตารางแม่ใหม่:** แก้ที่ `Search Name.xlsx` -> generate `products.json` ใหม่ -> อัปทับไฟล์เดิมบน GitHub

โครงสร้างข้อมูลแต่ละสินค้าใน `products.json`:
```json
{
  "sn": "กันแดดกันน้ำ 50ML",          // Search Name (ชื่อมาตรฐาน)
  "barcode": "8859572801513",          // บาร์โค้ด
  "watsons": "311957",                  // รหัสร้าน
  "erp": "HHFGSCWRSS0001",              // รหัส ERP
  "alias": ["311957"],                  // รหัสอื่นที่เจอใน CSV
  "names": ["Hexapeptide Sunscreen"],   // ชื่อใบเสร็จ (ค้นเจอด้วย)
  "desc": "HER HYNESS HEXAPEPTIDE...",  // ชื่อเต็มภาษาอังกฤษ
  "type": "NORMAL", "src": "master"
}
```
